# Release Notes - Shima v2.0.3

This release focuses on hardening the **Panel Control Agent** integration and providing a more robust, user-friendly experience when configuring complex workflows.

## Key Improvements

### 1. Fixed "AttributeError: FUNCTION" 🛠️
Resolved a critical issue where several panel nodes (Control Agent, Sampler, etc.) were missing essential metadata, causing ComfyUI's execution engine to crash. All panel nodes have been "hardened" with explicit execution paths.

### 2. Universal Architecture Protector 🛡️
Implemented a friendly validation system in the **Master Prompt** to prevent the cryptically named "y is None" and "shape mismatch" crashes.
- **Friendly Error Messages**: If your `model_type` doesn't match your loaded model, Shima will now tell you exactly what's wrong instead of crashing silently.

### 3. Panel Control Agent Polish 🏝️
- **Integrated Switches**: Removed redundant UI elements in favor of integrated hardware-style switches.
- **Autoconnect Ready**: Renamed key inputs to follow the Shima Bundle (`.bndl`) routing system for seamless auto-connections.
- **Improved Layout**: Adjusted node dimensions to perfectly fit the modular Island ecosystem.
