# Shima v2.0.12: Advanced Stealth Color Engine & Hot-Reloading 🎨🥷

This release finalizes the complete transition to the **Shima 3-Tier Semantic Color Engine**, effectively moving away from unstable canvas DOM hacks and directly integrating with ComfyUI's native LiteGraph API for flawless stealth rendering. We've also dramatically simplified the local development pipeline for theme designers.

## 🚀 Key Improvements

### 1. The 3-Tier Semantic Engine (NEW)
- **Full Color Engine**: Nodes take on vibrant semantic colors while text automatically mathematically calculates the best high-contrast foreground color (Black/White) for absolute readability.
- **Stealth (Dark Mode)**: Forces all node chassis to `neutral_dark` (#191919) but pushes the vibrant semantic category color purely into the title bar and editable text.
- **Stealth (Light Mode)**: Translates the stealth architecture to bright interfaces, utilizing `neutral_light` (#dddddd) chassis with vibrant text colors.
- **Permanent Node Serialization**: Stealth data overrides are beautifully mapped directly onto standard `node.properties`, ensuring your aesthetic customizations permanently survive ComfyUI reloads and workflow browser refreshes.
- **Native Context Menus**: The *Shima Colors* right-click menu has been refactored to inject custom palette selections natively, co-existing peacefully with the Stealth processing pipeline.

### 2. Live Excel Hot-Reloading
- Added a robust **📚 RELOAD THEMES (XLSX)** hardware button natively mapped into the chassis of the **Shima Custodian** node. 
- Integrated a new secure Python API endpoint (`/shima/settings/reload_themes`) to immediately purge the memory cache and re-parse your `shima_sheets.xlsx` spreadsheet from your disk. Theme designers can now infinitely tweak HEX codes in Excel and instantly spawn them in the canvas without ever violently terminating the ComfyUI server.

### 3. Stability & Housekeeping
- **Static API Namespace**: Successfully migrated all root `/extensions/Shima/*` HTML portals directly into the much cleaner, secure `/shima/*` endpoint prefix (e.g. `http://localhost:8000/shima/excel-editor.html`).
- **Alphabetized Settings Menu**: We've standardized our internal registry identifiers, effectively forcing the primary ComfyUI Vue settings menu to automatically list Shima configuration categories in alphabetical perfection.
