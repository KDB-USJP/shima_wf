# 🎨 Shima Styler: User Guide

Shima Styler is a suite of custom nodes for ComfyUI designed to make style selection, management, and experimentation effortless. Whether you're a power user building complex conditioning chains or just want to quickly try out different artist styles, Shima has a node for you.

## 1. Shima Style Selector (The Powerhouse)

The **Selector** node allows you to browse, search, and combine styles from a vast library of Artists and your own custom User Styles.

### Key Features
*   **Hybrid Output**: Generates both simple **Strings** (for basic CLIP encoding) and **Conditioning** (for advanced chaining).
*   **Visual Browser**: Filter by Category (e.g., "Anime", "Oil Painting") or Source (Artists vs. User Styles).
*   **Stackable**: Combine multiple styles with custom connectors (e.g., " and ", " mixed with ").
*   **Active Switch**: Easily bypass the node without disconnecting wires.

### Inputs & Outputs
| Input | Type | Description |
| :--- | :--- | :--- |
| **active** | Boolean | **On:** Applies styles. **Off:** Passes inputs through unchanged. |
| **base_prompt** | String | Your main positive prompt. Styles are appended to this. |
| **negative_prompt**| String | Your main negative prompt. Style negatives are appended to this. |
| **clip** | CLIP | (Optional) Required if you want `CONDITIONING` output. |
| **conditioning_positive** | COND | (Optional) Input conditioning to add styles *to*. |
| **conditioning_negative** | COND | (Optional) Input negative conditioning to add styles *to*. |
| **mode** | Combo | **Single:** Clicking a style replaces the selection. **Stack:** Clicking adds to the selection. |

### How to Use
1.  **Basic (String Mode)**: Connect your positive prompt to `base_prompt`. Select a style. Connect `positive` (String) output to a standard CLIP Text Encode node.
2.  **Advanced (Conditioning Mode)**: Connect your Loaded CLIP to `clip`. Connect `conditioning_positive` from a previous node (optional). The Selector will output ready-to-use `positive_conditioning` for your KSampler.

## 2. Shima Style Iterator (The Experimenter)

The **Iterator** node automatically cycles through a list of styles with every generation. Perfect for "XY Plot" style comparisons or generating large batches of variations.

### Key Features
*   **Auto-Queue**: Can automatically trigger the next generation in the queue until the batch is complete.
*   **Index Control**: Increment, Decrement, or Randomize the selection order.
*   **Scoped Batching**: Keeps track of its own index independent of global seed.

### Inputs
*   **style_mode**: Choose to iterate through "Artists" or "User Styles".
*   **index_mode**: How to pick the next style (Increment, Decrement, Randomize).
*   **batch_size**: How many styles to process before stopping (if auto-queue is on).

## 3. Shima Style Gallery (The Visualizer)

The **Gallery** node provides a grid view of styles with thumbnail previews.

### Key Features
*   **Thumbnails**: See a preview image for every artist/style.
*   **Grid Layout**: Easier browsing for visual learners.
*   **Search & Filter**: Same powerful search tools as the Selector.

## 4. Shima Favorites (The Shortcut)

The **Favorites** node is a lightweight text-based selector. It takes a list of IDs (e.g., `A1, U5`) and generates the prompts.

### How to Use
1.  In the **Selector** or **Gallery**, select your desired styles.
2.  Click the **"Copy IDs"** button (if available) or note the IDs (e.g., Artist #1 is `A1`, User Style #5 is `U5`).
3.  Paste them into the `favorites_list` box: `A1, U5, A100`.
4.  The node will generate the combined prompt. Great for saving specific combos without the full UI overhead.

## Tips & Tricks
*   **User Styles**: You can add your own styles! Edit the `shim_sheets.xlsx` file in the `assets/data` folder.
*   **Bypassing**: Use the `active` switch on the Selector to quickly compare "with style" vs "raw prompt" without rewiring.
*   **Negative Prompts**: Styles can contain negative prompts too! The Selector automatically combines them with your input negative.
