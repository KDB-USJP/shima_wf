# Shima v2.0.5: Dependency System Restoration 🏝️📦

This update restores core dependency management features and optimizes asset integrity checks.

## 🚀 Key Improvements

### 1. Dependency Generator & Installer
- **Restored**: Re-enabled the `Shima.DependencyGenerator` and `Shima.DependencyInstaller` nodes in the Python backend.
- **Workflow Audit**: The Generator now correctly scans prompts for model dependencies and outputs a JSON stub for sharing.
- **Batch Installation**: The Installer allows users to batch-download missing models directly to their local folders.

### 2. Asset Integrity & Performance
- **Dynamic Size Checks**: The `AssetManager` now performs a remote `HEAD` request to verify file sizes before downloading, preventing corruption and redundant data transfers.
- **Enhanced Redirect Support**: Improved handling of authenticated downloads from HuggingFace and Civitai.

### 3. UI Refinements
- **Colorized Output**: The Dependency Generator JSON output now features pink syntax highlighting for action items.
- **Node Polish**: Adjusted default sizes and layout for generator/installer nodes to improve workflow readability.

## ✅ Verification Complete
All restored components have been manually tested and confirmed functional in ComfyUI environments.
